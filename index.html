Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Class can be applied to multiple elements that will have the same CSS properties
  • Id can only be applied to one element and will have unique CSS properties

CSS

  • * is for all the properties
  • multiple sections can have the same CSS properties by using a comma to apply them
  • to assign properties to a class use a .
  • to assign properties to an id use a #

Git

  • Used to store, share and collaborate on work
  • git commands on the terminal always start with git

JavaScript

  • Conditional Statements: if, else if, else; make a decision about a piece of code
  • Loop statements: for(); going over pieces of code over and over again
  • Functions dont execute automatically, they must be called by name
  • A varaible is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
